# Damping of Space Plasma
Plasma Exploration Project - Plasma Physics 525 Spring 2026

## Basis
This project is based on _A Prescription for the Turbulent Heating of Astrophysical Plasmas_ by Greg Howes(2010) which aims to calculate relative heating of ions and electrons due to kinetic dissipation. This will be done with Plasma in a Linear Uniform Magnetized Environment(PLUME) https://kgklein.github.io/PLUME/. PLUME is a code deveolepd by Howes and others to numericaly solve the Vlasov-Maxwell equation in hot magnetized plasmas.
